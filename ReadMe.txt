Neste projeto será colocado tudo sobre o python3, é fundamental um sistema de leitura de arquivos para nosso
save ou que tenha um banco de dados. Espero fazer com muita organização e de forma responsiva.

O sistema de organização irá funcionar através de scripts separados por funções sem comunicação com outros
scripts sem ser no loop principal, em outras palavras possuirá independência para seu funcionamento, mas terá
scripts públicos com caractericas que possuirá em todos os scripts ou sua grande maioria, não queremos uma interface
gráfica é apenas um projeto para reforçar nossos conhecimentos de python3.

João Victor vulgo Gaguin deve fazer todos os sistemas lógicos deste rpg, e sua historia.
